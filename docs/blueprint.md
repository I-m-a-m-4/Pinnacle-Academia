# **App Name**: Zeneva

## Core Features:

- Multi-Tenant Inventory Management: Manage products with businessId association, including CSV import/export and product variants.
- Point of Sale (POS) System: Guided POS workflow with cart management, payment processing, customer selection, and receipt generation.
- Receipt Generation and Printing: Generate visually appealing, printable, exportable, and shareable receipts.
- Admin Settings: Manage business details, financial settings, subscriptions, and user access with password protection.
- Inventory Troubleshooting: Tool to check product price, missing categories, or potential duplicates. Suggest improvements for inventory data and merchandising.
- Product Variant Handling: Group products with the same name but different SKUs for better visibility.
- User and Role Management: Manage user profiles with businessId association and roles (admin, manager, vendor_operator).

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5), a calm and professional hue inspired by the need for clarity in complex business applications.
- Background color: Light blue (#E3F2FD), almost desaturated sky blue provides a clean, unobtrusive background.
- Accent color: Purple (#7E57C2), is an analogous color, creating a high contrast with the primary hue and visually highlighting key actions.
- Headline font: 'Space Grotesk' sans-serif font for headlines and short amounts of text; Body font: 'Inter' for longer text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist icons to represent inventory items, sales actions, and settings.
- Maintain a clean and organized layout with clear information hierarchy for ease of use.
- Use subtle transitions for data loading and form submissions.
